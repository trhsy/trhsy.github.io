            # 📚常见问题

            # 怎么反馈，提问问题？

            1. 提问之前，先去官网、文档、群文件看看，要搜寻对自己有用信息的能力，而不是啥都要问。
            2. 别人没有义务回答你，你要做好没人回答冷场的心里准备。
            3. 该有的礼貌要有，不要认为自己是大爷。
            4. 反馈问题，应该准备好: 遇到问题的详细场景，问题描述，截图或者录制的视频（若没准备好，切勿打扰被人）
            5. 注意表达、用词，例如：×这咋整？√大佬我这样做是不是哪里错了，为啥报错了？

            # 常见问题一 导入整合包

            我下载的整合包，群里下载了启动器为啥出现这种情况： 这种问题是你在导入过程中，因为网络等各种原因，forget等一些相关配置未下载成功，请重新导入，直至成功。 或者你可以开一下加速器，亦或者你可以改一下你的DNS

            # 常见问题二 蓝图问题

            为啥房子选项突然就没了？？？ 一看图中的模组就不是最新的，因为最新的模组是有标志性的↓↓↓↓↓

            建议更新到最新版本模组，当前1.7.10版本的模组最新是1.0.5

            如果更新的模组还是出现这个问题 1.检查mc-dushi-1.7.10\.minecraft\versions\trhsy-Sim-U\mods\sim\buildings下是否有内容 没有的话，检查网络（最好开加速器），重启游戏，因为建筑蓝图我放在了github服务器，所以你要有心里准备

            如果你这里面有东西，那你刚进游戏没有选项，请耐心等一分钟，文件太大需要加载一会会儿！

            # 常见问题三 游戏花屏

            如果你打开游戏之后花屏，拿东西啥的花屏请看下面解决方案，如果不想没辙了！！！ 打开查看图片 (opens new window)

            1. 检查自己的java版本（有的朋友说重装了java好了，最主要的是需要配置环境变量不会的找度娘去）
            2. 检查启动器版本（启动器尽量用hmcl）
            3. 用排查法，排查问题所在
              • 先开一个纯净mc1.7.10,启动游戏看看花屏不（win11这个步骤应该花屏,直接看第四步，设置显卡）
              • 再加入forge ,启动游戏看看花屏不
              • 之后再加入模组,启动游戏看看花屏不
            4. 有的玩家通过调整显卡成功了，不管是win10还是win11都可以通过这种方法，检查你的显卡设置 网友提供打开 如何设置显卡 (opens new window)

            # 常见问题四 游戏花技巧总结

            1.如果你已经开过一次档，删档重新开档的情况下，建议删档后，重启启动器再建档（因为你在删档后）有些配置文件并没有彻底删除，需要关闭启动器才能删。
