                      模拟城市相关教程

                      # 📚模拟城市

                      # 介绍

                      开始，你只有10金。

                      你可以雇佣NPC为你工作。

                      你的第一件事是雇佣一个NPC来建造一所房子，然后他们会搬进来，开始每天向你支付租金。

                      然后你可以用这笔钱雇佣更多的人来建设一座大城市。

                      完成你想在这个地方当镇长的梦想！

                      最高可雇佣200位NPC为打工，他们可以是建筑师，规划师，面包师，农民，屠夫，牧羊人，养鸡农民，养牛人，士兵，玻璃制造商，板砖工人等等很多角色。

                      这个模组最开始你只有10金，想要获得更多金币

                      第一步你要有一个可以雇佣工人的工具就是 建筑盒，采矿盒，养殖盒

                      建议先合成建筑盒，给工人建筑一些房子，最初先建一些便宜的房子，不然你容易资金亏损，到你资金为负的时候你房子还没建成，那你就完了

                      以上提到的资金来源有以下几种：

                      建房子，工人入驻，收租

                      商业经营所得

                      雇佣士兵，打怪物所得

                      建造建筑商，向建筑商卖东西所得

                      至于怎么建，在你正前方放一个建筑盒，右边放一个箱子

                      然后对着建筑盒右键你就懂了

                      采矿盒和养殖盒需要三个标记棒规划出一片区域才能放置，最后放置方法与建筑盒同理，简单介绍入门级教程到此结束。

                      # 物品合成篇

                      1. 建筑盒
                      2. 采矿箱
                      3. 养殖箱
                      4. 标记棒
                      5. 灯箱

                      # 用法篇

                      1.建筑箱：任意在一片空地放置，在左右两边任意放置箱子，对着建筑箱右键根据提示，可以雇佣NPC进行一些建筑/土地规划操作。

                      2.标记棒：可以由是三个标记棒划分出一片区域供采矿箱/养殖箱使用；也可以单独再任何一个箱子旁边放置，并空手对着标记棒右键可以标记地点，在控制箱中进行定点传输。标记棒在1.8.9版本新增了复制建筑成为蓝图的功能，用法是三个标记棒把建筑围起来，右键第一个标记棒，保存/复制建筑。这个时候你就能在你本地游戏目录下/mods/sim/buildling/other/my开头的TXT文件，为你的建筑重新命名，你就能再重启游戏后让NPC为你建筑你保存的蓝图了，如果你想让你的建筑加入到模组里，可以联系作者。

                      3.采矿箱：必须依靠标记棒划分出一片区域才能使用；用法请参考图3-4。

                      4.养殖箱：必须依靠标记棒划分出一片区域才能使用；用法请参考图3-4。

                      5.灯箱：灯箱正常的照明方块，可以当灯泡用，当然没有火把使用，但是比火把好看。 图：3-4

                      # 建筑介绍

                      建筑箱雇佣好建筑工之后，会出现一下四个选项框：

                      住宅，商业，工业，其他。 顾名思义：主要的是住宅，你想要NPC为你卖命总要给人家提供住的地方，第一个建筑肯定是住宅（经验之谈，前期请先建筑低资金的建筑。）

                      商业：

                      1.杂货店（水果店更合适）：你可以提供一些水果放到箱子里人npc帮你赚钱

                      2.面包店：你可以直接放面包进去让NPC帮你卖，也可以用农田箱建筑一个小麦农场，让NPC自给自足。

                      3.肉铺：卖一些生肉给小镇的居民

                      4.诊所：用来让女性NPC生孩子用的。

                      5.银行：存放贵重物品，暂未修复，无法使用

                      6.汉堡店：在后厨箱子里放入面包，奶酪，生牛肉，马铃薯。能够做出来，汉堡，炸薯条，奶酪汉堡。 工业：

                      1.板砖厂：雇佣NPC后会在附近寻找黏土球，然后放到熔炉里烧制砖块

                      2.玻璃工厂：雇佣NPC后会在附近寻找沙子，然后放到熔炉里烧制玻璃

                      3.仓库：雇佣NPC后可以使用路径箱标记一个有箱子的地方让NPC把箱子内的东西搬运回仓库

                      4.伐木场：雇佣NPC后可以在附近寻找树，获取到木材放到伐木场的箱子里。

                      5.鸡蛋农场：雇佣NPC后自动生成6只鸡，获取鸡蛋（用于面包店做蛋糕）

                      6.建筑商：雇佣NPC后可以买卖建筑建材

                      7.军营：可以雇佣最高10个NPC成为战士为你巡逻村子

                      8.牧羊场：可以雇佣NPC养羊获取羊毛

                      9.奶酪工厂：可以雇佣NPC从奶牛场获取牛奶，制作奶酪块

                      10.奶牛场:可以雇佣NPC养奶牛获取牛奶

                      11.养鸡场:可以雇佣NPC养鸡获取鸡肉

                      12.养牛场:可以雇佣NPC养牛获取牛肉

                      13.养兔场:可以雇佣NPC养兔子获取兔肉

                      14.养猪场:可以雇佣NPC养猪获取猪肉

                      15.渔场:可以雇佣最多两个NPC钓鱼获取鱼

                      # 更新篇

                      1.8.9版本又新增了很多商业建筑蓝图（宠物店，电影院，花店，火车站，麦当劳，食品店（原杂货铺），酒馆，医院，杂货铺），虽然目前只是开发阶段但是已经可以建造了敬请期待吧 最新更新，所有建筑蓝图将不再从github下载，直接嵌入到模组里，1.7.10从1.0.6版本开始；1.8.9从1.0.5版本开始。

                      # 替换模组

                      去下载最新的模组，切记下载完最新的模组后把模组替换的/mods目录下，并删除sim文件夹重启游戏。 下载模组时，请找到最新的版本一般都是1.0.X,最新的模组（修复率是最高的） 下载最新模组 打开你的启动器，找到整合包并打开

                      # 1.8.9整合包/模组通过HMCL启动器下载

                      # 整合包

                      1. 首先打开启动器，选择"版本列表"

                      1. 点击"安装新游戏版本"-"整合包"，在搜索框输入"sim-u",点击【搜索】

                      1. 选择合适的版本，点击下载就行了，选个地址保存本地就行了

                      # 模组

                      同整合包一样，点【模组】输入【sim-u】点【搜索】就能找到模拟城市的模组了

                      跟整合包下载一样点击下载，选个地址保存，然后复制的/mods/目录下
